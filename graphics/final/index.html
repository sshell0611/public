Final Project: HammerVis

Project Scope

I would like to incorporate some of the graphics and animation concepts we have used throughout the semester into my own data visualization package. The project will explore the overlap of data analytics and computer graphics. The animation used in the charts will not be gratuitous, but rather, will enhance the viewer's understanding of the data. Graphics techniques will be used as a way to coherently fit more "dimensions" of the data into the small space available on a webpage. Specifically, here are a couple ideas for charts: 1) Bubble charts usually show 3 dimensions of data using the x and y axis and size/weight of the bubble. I will incorporate additional dimensions of the data using textures and animation. 2) 3D surface modeling using splines. Incorporate a time-lapse component using animation to show changes in the surface. The concepts from class I will use are: - Parametric shapes - Procedural texture - Lighting/Colors/Shading - Movement and Rotation - Splines - Javascript - three.js
